FINDING KEY STATE OF EACH TURN

Found the Key state of each turn by implenting two methods

  1. Finding the intent using BERT
  2. Implented chatbot using gobot model in Deeppavlov

Intent Classifier Using BERT
BERT(Bidirectional Encoder Representations from Transformers) model made possible of achieving State-of-the-art result for finding the key state of the conversation.BERT is built on top of multiple clever ideas by the NLP community. Some examples are ELMo, The Transformer, and the OpenAI Transformer. We have used BERT because it looks both forward and backward so we get best result and it can be best used for small amount of data also.This model will classify two intents based on the dataset provided.


Intents classified are:
text: What's the best Wireframing tool? intent: System Specification
text: did you install just the jre? intent: Settings
text: I use IBM's 1.4.2 intent: System Specification

Implented chatbot using gobot model in Deeppavlov
Deeppavlov is a framework which can be used to build models.Among them is gobot model which can create a contextual chatbot.Contextual chatbot are created with the help of intent. This chatbot will reply to the user according to intent which will make the conversation between the user and bot more contextualized.
The before model which found the intent with the help of BERT is implemented in gobot model by using RNN approach.In gobot model the dataset is a labelled data ,so that the the conversation between the user and the bot is improved.
